# scripts/analysis/ — 结果分析

对 Judge 结果进行统计分析和质量检查。

## 文件说明

| 文件 | 说明 |
|------|------|
| `analyze_judge_results.py` | Judge 结果汇总分析（AC 率、状态分布等） |
| `analyze_ce_errors.py` | CE 错误深度分析（原因分类、样例提取） |
| `check_output_format.py` | 检查模型输出格式是否合规 |
| `generate_final_stats.py` | 生成最终统计报告 |
| `_tmp_progress.py` | 临时进度统计脚本 |
